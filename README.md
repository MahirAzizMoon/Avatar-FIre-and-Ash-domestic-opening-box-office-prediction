# Avatar: Fire and Ash domestic opening Week Box Office Revenue Prediction

This project builds an end-to-end machine learning pipeline to predict the **opening domestic week box office revenue** of an unreleased film, using *Avatar: Fire and Ash* as a case study.

## What I Did
- Scraped historical movie metadata using the **TMDB API**
- Combined metadata with historical box office revenue
- Cleaned and engineered features suitable for pre-release prediction
- Trained and evaluated multiple regression models
- Estimated prediction uncertainty using historical error patterns
- Generated a baseline opening-week revenue prediction for the upcoming Avatar release
